# 文化大図鑑（仮）

**URL**: [https://museum.jackotonashi.com](https://museum.jackotonashi.com)

## 概要

武蔵野美術大学通信教育課程に在籍しながら、自分が興味を持ち続けられるよう制作しました。ムサビ通信の課題を進めるたびに更新されますが、関係ないネタも随時更新しています。

## 🛠️ 技術スタック

- **フレームワーク**: React 18.3.1
- **言語**: TypeScript
- **ビルドツール**: Vite 6.3.5
- **UI ライブラリ**: Material-UI 7.3.5
- **スタイリング**: Tailwind CSS v4
- **フォント**: Noto Serif JP（Google Fonts）
- **デプロイ**: Vercel
- **バージョン管理**: Git / GitHub

## 🚀 開発

```bash
# 依存関係のインストール
pnpm install

# 開発サーバーの起動
pnpm dev

# ビルド
pnpm build
```
